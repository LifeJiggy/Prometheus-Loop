# Prometheus Loop

A comprehensive reference for building, teaching, and reasoning about agentic AI systems.

## What's Inside

- **Three maturity levels** — Concept, Production, Autonomous
- **13 self-* capabilities** — Each with 700+ lines of implementation
- **Plugin system** — Install to 18+ CLI/IDE tools
- **Documentation** — Guides, API reference, troubleshooting
- **Examples** — 8 case studies with real-world use cases

## Quick Start

```bash
# Install the plugin
bash prometheus-loop-plugin/scripts/install.sh --all

# Use in your CLI/IDE
/loop                    # Show overview
/loop self-healing       # Show specific capability
```
